Brands

Pick a brand from your calendar and set its pounds of grain per turn (per batch). The value saves on this device until you change it. Day and week totals multiply it by the turns on the sheet.

Auto‑sync Recommended

This app is pre‑connected to your shared Google Sheet — every device that opens the link reads and writes the same data automatically, so there's nothing to set up here. The URL below is baked into the app; you only touch it if you ever create a brand‑new deployment.

Manual data source Fallback

Point the app at your Google brewing calendar. For live reading, set the sheet to Anyone with the link → Viewer (or File → Share → Publish to web). If live loading is blocked, use Paste mode below — copy the cells straight out of Google Sheets and paste them in.

Paste mode (fallback) — click to open

Select your calendar cells in Google Sheets, copy, and paste here. Keep the header row.
